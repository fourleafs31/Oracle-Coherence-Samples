This is simpel coherence App test.

* Requirement
1. ant
2. JDK
3. Coherence

* How to build.
1. Modify the following files for your enviroment.

setEnv.sh
src.properties

2. Run the below:

$ . ./setEnv.sh

3. Run the below:

$ ant

* How to run.

0. Create ./logs directory:

$ mkdir logs

1. Simple App: 

$ ./HelloWorld.sh
